# Branding Assets

Generated placeholder release assets for Career Chaos Academy.

Files:

- `app_icon.png` - launcher icon source
- `app_icon_foreground.png` - Android adaptive foreground
- `splash_logo.png` - splash logo
- `splash_logo_android12.png` - Android 12 splash logo

Before final production release, replace these with professionally designed assets if needed, then run:

```bash
dart run flutter_launcher_icons
dart run flutter_native_splash:create
```
