# Privacy Policy

**App name:** Career Chaos Academy  
**Package name:** com.careerchaos.academy  
**Effective date:** 16 June 2026

Career Chaos Academy is an educational role-based learning game. It uses humorous workplace scenarios, mini-games, scores, badges, and progress tracking to help users practise decision-making.

## Information the app may store

The app may store the following information locally on the user's device:

- Completed chapters
- Unlocked chapters
- Role-wise score
- Total XP
- Career rank
- Badge progress
- Mini-game completion status
- Theme preference

This local data is used only to restore progress and improve the user experience.

## Backend account and progress data

When the optional backend API is enabled, the app may send and receive progress information using a generated device/user identifier. The backend may store:

- User/device identifier
- Role progress
- Completed chapters
- Scores and XP
- Badges
- Mini-game progress

The app should not collect unnecessary personal information for gameplay.

## Analytics and crash reporting

For release builds, the app includes optional monitoring hooks:

- Crash reporting can be enabled using Sentry.
- Analytics can be enabled using Firebase Analytics.

These services are disabled by default in the source configuration and must be explicitly configured with release environment values before production use.

If enabled, analytics and crash reporting may collect technical diagnostics such as app version, device type, crash stack traces, and usage events such as chapter completion. Do not log sensitive personal information, passwords, payment details, medical details, or private user notes.

## Professional advice disclaimer

Career Chaos Academy is for education and entertainment. It does not provide professional medical, legal, financial, engineering, safety, HR, or compliance advice.

Medical, legal, and financial scenarios must remain general and should encourage safe professional consultation where required. The app must not be used for diagnosis, prescription, legal decision-making, investment advice, or emergency guidance.

## Children and students

The content is designed for general learning. If the app is offered to children or students, the publisher should review local privacy requirements and store listing rules before release.

## Data deletion

If the app stores progress locally, users can reset progress from inside the app. If a backend server is enabled, the publisher should provide a support channel for backend progress deletion requests.
