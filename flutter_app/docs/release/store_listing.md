## Full description

Career Chaos Academy turns workplace learning into a funny interactive game.

Choose a profession, enter chaotic real-world-inspired scenarios, make choices, solve mini-games, earn XP, unlock badges, and grow from Intern to Legend.

Play as roles like Developer, QA Tester, Doctor, Civil Engineer, Architect, Back Office Executive, Project Manager, and HR Executive. Each role has story chapters, funny consequences, professional learning points, and decision-based outcomes.

Key features:

- Role-based career scenarios
- Multi-chapter progression
- Choice-based outcomes
- Mini-games such as code fix, multiple select, match pairs, arrange order, data cleanup, and decision matrix
- XP, badges, and rank progression
- Local progress saving
- Optional backend sync
- Admin CMS for scenario publishing
- AI-assisted scenario review workflow
- Dark mode and responsive UI

Career Chaos Academy is designed for learning, entertainment, and career awareness. It does not provide medical, legal, financial, engineering, HR, or compliance advice.
